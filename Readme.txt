== Description ==

This Plugin mission is to correct up to 15 words it's meaning is to say "If Allah Wanted" but with a wrong phrasing .

مهمة هذه الإضافة هي تصحيح كلمة "إن شاء الله" باستبدال 15 طريقة خاطئة لكتابتها .

1. This section describes how to install the plugin and get it working.
